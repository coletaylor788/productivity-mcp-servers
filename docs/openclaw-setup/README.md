These are step-by-step guides from my jour build a personal AI agent ("Puddles") on a Mac Mini, from a fresh box to a fully running, remotely-managed assistant with native Apple integrations.

These are reference docs you can follow with no prior context — each one assumes only what the previous guide produced.
